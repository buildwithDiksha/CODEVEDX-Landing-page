# TechNova — AI Startup Landing Page

> A clean, responsive landing page for an AI startup, built entirely with HTML5 and CSS3.

🔗 **Live Demo:** [buildwithdiksha.github.io/CODEVEDX-Landing-page](https://buildwithdiksha.github.io/CODEVEDX-Landing-page/)

---

## 📌 Overview

**TechNova** is a modern, single-page website designed for an AI technology startup. The project demonstrates professional front-end layout and design skills using only HTML5 and CSS3 — no JavaScript frameworks, no external libraries, no dependencies.

It was built as part of the **CODEVEDX** portfolio project to showcase real-world UI design capabilities through clean semantic markup and responsive styling.

---

## ✨ Features

- **Responsive Navigation Bar** — Sticky navbar with smooth anchor links to all sections
- **Hero Section** — Eye-catching headline with dual call-to-action buttons and a hero image
- **Features Section** — Highlights three core AI capabilities: Smart Automation, Data Analytics, and Cloud Security
- **About Section** — Company mission statement with a supporting team visual
- **Contact Section** — Simple contact form with a message input and submit button
- **Footer** — Clean copyright footer

## 🚀 Getting Started

No installation or build step required. Just open the file in a browser.

### Option 1 — Open Locally

```bash
# Clone the repository
git clone https://github.com/buildwithdiksha/CODEVEDX-Landing-page.git

# Navigate to the project folder
cd CODEVEDX-Landing-page

# Open in browser
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

### Option 2 — View Live

Visit the live deployment directly:
👉 [https://buildwithdiksha.github.io/CODEVEDX-Landing-page/](https://buildwithdiksha.github.io/CODEVEDX-Landing-page/)
